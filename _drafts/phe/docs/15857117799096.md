# 2020-4-1-QGIS 影像几何校正及数字化
reference: https://malagis.com/qgis-handbook-7-1.html

影像校正(GeoReference)指的是将无坐目标影像透过参考图层建立控制点，将其赋予坐标。有许多影像拥有很多有用的信息，但却没有坐标信息，若能赋予这些影像坐标，将可大幅提升其使用效用。
